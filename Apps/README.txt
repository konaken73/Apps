cette application permettant d'éffectuer  rechercher les informations d'un pays en fonction de :
   - le nom du pays
   - le code telephonique
   - la capitale .
   
  Ainsi retourne les informations concernant le dit pays  à savoir :
  
    - nom, capitale , device monetaire ,langues officielles, zone geographique ,zone geographique precise , code telephonique ,population .
    
    
    # prerequis 
    
    - installation de node.js
    - installation des paquet express  ( 
    
     * cd  RepertoireApplication
     * npm install express
     
     )
     
     # utilisation 
     
     - cd  RepertoireApplication
     - node appRest
